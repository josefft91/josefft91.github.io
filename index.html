  • Incrementally optimised the website for mobile, which has led to an increase in mobile traffic.

Compliance Control Ltd

As part of my degree I did an industrial placement at a company called Compliance Control Ltd. My main job for Compliance Control Ltd was furthering the development of a piece of software called "ComplianceControl Centre", which is an Electronic Document Management System and Electronic Quality Management System. The software was written using PHP with a Model-View-Controller architecture with an Oracle database as the backend database. The software is currently being used by several companies around the world.
